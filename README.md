## Setup

```sh
# Install dependencies
$ pnpm install

# Compile and hot-reload for development
$ pnpm dev

# Compile and minify for production
$ pnpm build

# Lint with eslint
$ pnpm lint

# Run vitest tests
$ pnpm test
```
